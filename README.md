# Digital Resume Builder & Profile Management System

A full-stack web application for creating and managing professional resumes online.

## Tech Stack
- **Frontend:** React.js (Vite)
- **Backend:** Java Servlets on Apache Tomcat
- **Database:** MySQL
- **Deployment:** AWS (EC2, RDS, S3)

## Features
- User Registration & Login
- Profile Management
- Skills, Projects, Education, Experience, Certifications
- Real-time Resume Preview
- One-click PDF Download
- Shareable Profile Link
- Admin Control Panel
